

## Getting Started
First, install the node module:

npm install

 run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
##  Node Version
Node.js: v20.19.0

## Next js Version
Next.js: v16

## Build the Development Version
npm run build

## After build run this command
npm start


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
